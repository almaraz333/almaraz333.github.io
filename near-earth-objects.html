A Visualization of Near Earth Objects from NASA's API



An interactive display of the 10 nearest objects to Earth using React and the official NASA API.




Project Overview


Challenge:

Create an interactive visualization of objects that are going have near earth orbits using React and pulling the data from the NASA API while maintaining a good user experience and as much accessibility as possible.

Team:

Myself


The Process



What is the Goal?

The goal for this project was to create a React app that displays near earth objects that have the highest probability to hit Earth to learn React and show an understanding of the language. This app could also serve as a learning tool for kids who have an interest in space.




Necessities:


  • Efficient
  • Minimal
  • Kid Friendly
  • USer Friendly
  • Good use of React and States


Learning React

Learning a coding language is similar to learning a regular language. From an outsider looking in it makes no sense. This is how I felt when I first took on the task of learning React. The main sources I used to learn React were the official documentation and Youtube videos. After hours of trial and error, frustration, and StackOverflow I was able to get a relatively solid handle on the language and create the thing I set out to.




What I Learned

  1. React is a versatile language and can do a lot. At first it was hard to understand but after some practice and many Youtube videos I got a solid understanding of it and am now able to use it to build apps and sites.

  2. Learning a new language is the same every time, whether it's coding or not. It takes a lot of practice to even start to understand what is really going on behind the scenes instead of memorizing expressions.

  3. As a developer I am never done learning. Especially with the exponential way the field expands there is always something to learn and better yourself with. That is why I make sure I never go stale and stop trying to perfect the way I work.
